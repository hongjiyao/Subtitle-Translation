# 视频转字幕工具

一个功能强大、界面友好的视频转字幕解决方案，支持多语言识别和翻译，基于先进的AI模型提供高质量的字幕生成。

## 功能特点

- 🎯 **多语言支持**：支持15种语言的自动检测和手动选择
- 📹 **批量处理**：支持多文件上传和队列管理
- 🔍 **高精度识别**：使用WhisperX模型进行语音识别，提供更准确的时间戳
- 🌐 **智能翻译**：使用腾讯HY-MT1.5-7B-GGUF模型进行高质量翻译
- ⚙️ **丰富的参数设置**：可调整模型大小、批处理大小、VAD过滤等参数
- 💾 **参数保存**：支持保存和加载参数配置
- 🖥️ **友好的界面**：基于Gradio构建的现代Web界面
- 🚀 **性能优化**：支持GPU加速和模型量化

## 安装步骤

### 推荐安装流程

**无需配置任何环境，也无需手动下载任何程序，只需按顺序运行两个脚本即可完成所有安装和配置。**

1. **克隆项目**
   ```bash
   git clone https://github.com/yourusername/subtitle-translation.git
   cd subtitle-translation
   ```

2. **第一步：运行安装脚本**
   ```bash
   venv_setup.bat（用管理员身份运行）
   ```

   这个脚本会自动：
   - 清理旧的虚拟环境
   - 检查并安装Python 3.11.7（如果未安装）
   - 创建新的虚拟环境 `.venv_final`
   - 升级pip
   - 安装所有必要的依赖包（包括PyTorch、Gradio、Whisper等）
   - 验证安装

3. **第二步：运行启动脚本**
   ```bash
   start.bat
   ```

   这个脚本会自动：
   - 检查并安装Visual C++ Redistributable（如果未安装）
   - 检查Python环境
   - 提供菜单选项：
     1. 下载所有模型
     2. 运行UI
     3. 先下载模型再运行UI
     4. 退出

## 使用方法

### 推荐使用方法

**安装完成后，只需运行启动脚本即可开始使用：**

1. **运行启动脚本**
   ```bash
   start.bat
   ```

2. **选择菜单选项**：
   - 输入 `1` 下载所有模型（首次使用时推荐）
   - 输入 `2` 直接运行UI（会自动打开浏览器）
   - 输入 `3` 先下载模型再运行UI（首次使用时推荐，会自动打开浏览器）
   - 输入 `4` 退出

3. **浏览器会自动打开**：http://localhost:7860

4. **使用步骤**：
   - 点击"上传视频文件"按钮选择视频文件
   - 在"详细配置"标签页调整模型和参数设置
   - 点击"添加到队列"按钮将文件添加到处理队列
   - 点击"处理队列"按钮开始处理
   - 处理完成后，字幕文件会保存在`outputs`目录中

### 其他使用方法

#### 通过GUI界面使用

1. **激活虚拟环境**（如果还未激活）
   ```bash
   .venv_final\Scripts\activate.bat
   ```

2. **启动应用**
   ```bash
   python ui.py
   ```

3. **在浏览器中打开**：http://localhost:7860

## 示例

### 示例1：处理单个视频文件

1. **运行启动脚本**
   ```bash
   start.bat
   ```

2. 输入 `3` 先下载模型再运行UI
3. 浏览器会自动打开 http://localhost:7860
4. 上传视频文件
5. 选择模型和语言设置
6. 点击"添加到队列"
7. 点击"处理队列"
8. 处理完成后，在`outputs`目录中查看生成的字幕文件

### 示例2：批量处理多个视频文件

1. **运行启动脚本**
   ```bash
   start.bat
   ```

3. 浏览器会自动打开 http://localhost:7860

5. 选择统一的模型和参数设置
6. 点击"添加到队列"
7. 点击"处理队列"
8. 处理完成后，所有视频的字幕文件都会保存在`outputs`目录中

## 技术细节

- **语音识别**：使用WhisperX模型（基于OpenAI的Whisper）
- **翻译**：使用腾讯HY-MT1.5-7B-GGUF模型
- **UI框架**：Gradio
- **视频处理**：FFmpeg
- **队列管理**：自定义队列系统，支持实时进度更新
- **模型量化**：使用llama.cpp进行模型量化

## 注意事项

- 处理长视频可能需要较长时间
- 较大的模型会提供更高的准确率，但处理速度会更慢
- 翻译质量取决于源语言识别的准确性
- 建议在处理前预览视频，确保音频质量良好
- 首次使用时需要下载模型，可能需要较长时间
