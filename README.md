# 视频转字幕工具

一个功能强大、界面友好的视频转字幕解决方案，支持多语言识别和翻译。

## 功能特点

- 🎯 **多语言支持**：支持自动检测和手动选择多种语言
- 📹 **批量处理**：支持多文件上传和队列管理
- 🔍 **高精度识别**：使用Whisper模型进行语音识别
- 🌐 **实时翻译**：支持多种语言之间的翻译
- ⚙️ **丰富的参数设置**：可调整模型大小、批处理大小、VAD过滤等参数
- 💾 **参数保存**：支持保存和加载参数配置
- 🖥️ **友好的界面**：基于Gradio构建的现代Web界面

## 系统要求

- Python 3.8+
- 足够的内存（推荐8GB以上）
- GPU支持（可选，但推荐用于加速处理）

## 安装步骤

### 推荐安装流程

**无需配置任何环境，也无需手动下载任何程序，只需按顺序运行两个脚本即可完成所有安装和配置。**

1. **克隆项目**
   ```bash
   git clone https://github.com/yourusername/subtitle-translation.git
   cd subtitle-translation
   ```

2. **第一步：运行安装脚本**
   ```bash
   venv_setup.bat
   ```

   这个脚本会自动：
   - 清理旧的虚拟环境
   - 检查并安装Python 3.11.7（如果未安装）
   - 创建新的虚拟环境 `.venv_final`
   - 升级pip
   - 安装所有必要的依赖包（包括PyTorch、Gradio、Whisper等）
   - 验证安装

3. **第二步：运行启动脚本**
   ```bash
   start.bat
   ```

   这个脚本会自动：
   - 检查并安装Visual C++ Redistributable（如果未安装）
   - 检查Python环境
   - 提供菜单选项：
     1. 下载所有模型
     2. 运行UI
     3. 先下载模型再运行UI
     4. 退出


## 使用方法

### 推荐使用方法

**安装完成后，只需运行启动脚本即可开始使用：**

1. **运行启动脚本**
   ```bash
   start.bat
   ```

2. **选择菜单选项**：
   - 输入 `1` 下载所有模型（首次使用时推荐）
   - 输入 `2` 直接运行UI（会自动打开浏览器）
   - 输入 `3` 先下载模型再运行UI（首次使用时推荐，会自动打开浏览器）
   - 输入 `4` 退出

3. **浏览器会自动打开**：http://localhost:7870

4. **使用步骤**：
   - 点击"选择文件"按钮上传视频文件
   - 选择合适的模型和参数
   - 点击"添加到队列"
   - 点击"开始处理"按钮
   - 处理完成后，字幕文件会保存在`output`目录中

### 其他使用方法

#### 通过GUI界面使用

1. **激活虚拟环境**（如果还未激活）
   ```bash
   .venv_final\Scripts\activate.bat
   ```

2. **启动应用**
   ```bash
   python ui.py
   ```

3. **在浏览器中打开**：http://localhost:7870

#### 通过命令行使用

1. **激活虚拟环境**（如果还未激活）
   ```bash
   .venv_final\Scripts\activate.bat
   ```

2. **运行命令**
   ```bash
   python ui.py --model large-v3 --translator m2m100_1.2B --device cuda --source-language auto --target-language zh --video-files video1.mp4 video2.mp4 --process
   ```

## 参数说明

### 基本参数

- **识别模型**：选择Whisper模型大小（tiny, base, small, medium, large-v2, large-v3）
- **翻译模型**：选择翻译模型（m2m100_418M, m2m100_1.2B）
- **设备**：选择运行设备（auto, cpu, cuda）
- **语言模式**：选择语言检测模式（自动检测, 手动选择）
- **源语言**：选择源语言（auto表示自动检测）
- **目标语言**：选择目标语言

### 高级参数

- **语音模型批处理大小**：调整批处理大小以平衡速度和内存使用
- **翻译模型批处理大小**：调整翻译模型的批处理大小
- **beam size**：调整识别和翻译的beam search大小
- **VAD过滤**：启用或禁用语音活动检测
- **VAD参数**：调整VAD阈值、最小语音持续时间等参数
- **单词时间戳**：为每个单词添加时间戳
- **基于先前文本**：利用上下文信息提高准确率

## 输出文件

处理完成后，字幕文件会保存在`output`目录中，格式为`.srt`文件。

## 故障排除

- **内存不足**：如果遇到内存不足错误，可以减小批处理大小
- **模型下载失败**：确保网络连接正常，或手动下载模型到`models`目录

## 性能优化

- **使用GPU**：如果有GPU，选择`cuda`设备可以显著提高处理速度
- **选择合适的模型**：根据视频长度和质量选择合适的模型大小
- **调整批处理大小**：根据可用内存调整批处理大小

## 技术细节

- **语音识别**：使用OpenAI的Whisper模型
- **翻译**：使用Facebook的M2M100模型
- **UI框架**：Gradio
- **视频处理**：FFmpeg
- **队列管理**：自定义队列系统，支持实时进度更新

## 注意事项

- 处理长视频可能需要较长时间
- 较大的模型会提供更高的准确率，但处理速度会更慢
- 翻译质量取决于源语言识别的准确性
- 建议在处理前预览视频，确保音频质量良好
